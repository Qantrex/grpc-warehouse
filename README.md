## Implementierung

Start HelloWorldServer (Java)
`gradle clean build`
`gradle runServer`

Start HelloWorldClient (Java)
`gradle runClient`

For Python: Add grpcio packages
`pip3 install grpcio grpcio-tools`

Start HelloWorldClient (Python)
`python3 src/main/resources/helloWorldClient.py`
